원본 제목(영문): Building with Realtime Mini
번역 제목(한글): Realtime Mini로 구축하기
이 문서는 OpenAI Agents SDK와 최신 음성 모델(4o-audio, GPT Realtime Mini)을 활용해 지연이 낮고 도구 호출이 강화된 실시간 음성 고객지원 에이전트를 ‘handoff’ 아키텍처로 구현하는 방법을 설명합니다. 음성 에이전트 특성상 WebRTC 세션 관리, 음성활동감지(VAD)로 인한 변동성, 높은 지연 민감도를 다루는 설계 원칙을 제시합니다. 아키텍처는 메인 에이전트가 사용자의 의도를 분류해 일반 문의는 인증 없이 문서 조회 기반의 QA 에이전트로, 개인 정보가 필요한 요청은 인증을 거친 전문 에이전트나 도구로 안전하게 핸드오프하도록 구성됩니다. 인증 단계는 needsApproval 설정을 통해 UI에서 자격 증명을 수집하고 사용자 승인/거절을 받아 도구 호출을 제어하며, 전체 흐름은 실시간 세션 관리로 통합 운용됩니다. 프런트엔드는 승인 플로우, 세션 상태 관리, 음소거 및 음성·이미지 등 멀티모달 입력을 조율해 안정적이고 일관된 상호작용을 보장합니다.
